# Fiesto

Fiesto is a full-stack web-based food delivery application that focuses on providing users a seamless experience in ordering food from their favorite restaurants.

## Project Overview
Fiesto allows users to browse restaurants, view menus, place orders, track deliveries, and make payments conveniently through an intuitive interface.

## Features
- Browse and search restaurants
- View detailed menus
- Order food with real-time tracking
- User registration and login
- Payment integration

## Technology Stack
- **Frontend:** HTML (70.6%), CSS (19.4%), JavaScript (7.9%)
- **Backend:** Node.js, Express
- **Blockchain:** Smart contracts developed using Solidity (2.1%)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/SuryaMVN07/food-delivery-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd food-delivery-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```

## Usage
- Open the application in your browser at `http://localhost:3000`
- Follow the instructions to create an account and start ordering your favorite meals!

## Smart Contracts
Fiesto utilizes smart contracts to ensure secure transactions and order management on the blockchain. Documentation for the smart contracts can be found in the `/contracts` directory.
